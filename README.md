# ember-cli-test-helpers

## Description
Ember-cli-test-helpers is a collection of test helpers

## Installation
```
# install via npm
$ npm install ember-cli-test-helpers --save-dev
```

## Example usage of helpers can be found in usage-test.js
```
$ npm install
$ bower install
$ ember test
```
